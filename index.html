阿美族

概述

阿美族屬於母系社會，在其傳統社會中，家庭成員以女性尊長為家長，女孩子擁有繼承家產的優先權，而家庭對外事務則由家長之兄弟或舅父代為處理。這樣的社會特質，表現在服飾穿著的特徵上，每一年齡階級都有不同的服飾表現，代表著穿著者的身份與責任。

傳統服飾與配色

  1. 男子服飾：上半身穿著麻製上衣或者是皮衣，而下半身僅圍一條遮陰布。冬季時，會在外面加穿套袖或者是披掛上一件胸披，以抵禦寒冷。
  2. 女子服飾：上半身穿著對襟圓領長袖短上衣，下半身則圍上一或二條的圍裙，小腿部分常見綁繫一對護腳布。參加祭儀時，著盛裝且頭戴珠冠（或羽冠），紅色主要是年輕女子穿著，黑色則表示為年紀較長的婦女。

布農族

概述

布農族的分佈區域相當廣泛，重點區域包括南投縣、花蓮縣、臺東縣、高雄縣等，主要以打獵維生，在服飾和編織上，材料以及製作大都是自給自足的方式。

服飾材料來源

以早期衣服主要的原料是獸皮和麻布；棉、毛等材料，一直到與漢人接觸後陸續引進使用。

傳統服飾顏色搭配

  1. 因應他們狩獵生活的環境，色系偏暗色或深色，因太亮色在森林中，較容易被動物發覺，選擇暗沉的顏色易於隱藏身分。
  2. 當布農族男性成為族裡的打獵英雄之後，可以被允許穿上鮮豔的紅色。
  3. 女性服裝顏色常見深暗色調的藍、黑色，而服裝上的織紋較多用黃、紅、黑、紫四色交互搭配而成條紋。
  4. 在祭儀上穿著的傳統衣飾普遍是以白色為底，並且圖紋以直條人字紋為主，紋飾產生喜用紅、黃、紫或紅、黃、白搭配出來。

達悟族

概述

達悟族也稱雅美族，分布於臺東縣的蘭嶼島上，有豐富的神話傳說與歲時祭儀，並具有明顯的海洋特質。

服飾材料原料

達悟族是原住民中穿著最少的一群，服飾由天然麻、香蕉葉等植物纖維製作而成。

傳統服飾特色

  1. 男子傳統服裝以無袖無領的短背心，搭配上丁字褲，方便通風與捕魚活動。
  2. 女子穿著胸兜或是背心上衣，下半身則穿著繫帶的方布裙。
  3. 所有手織衣物均以白色為底，黑、藍相間的色彩。
  4. 重要喜慶儀式與場合時，男女穿著白底夾織藍紋的典服；男子配戴銀盔或藤盔，女子則是戴椰鬚帽或八角禮帽，另外再搭配金銀飾品裝飾。

邵族

概述

邵族主要居住在南投縣魚池鄉伊達邵部落和水里鄉大坪林聚落，其中伊達邵部落至今仍然嚴謹地遵循傳統祖靈信仰，並完整遵循每一祭典儀式的禮俗規矩。

衣料

邵族傳統的衣料有皮革、麻布、樹皮布和棉布四種。皮革多用於男子服飾，由男子自行揉製而成，而麻布則用於女子服飾，由女子自織苧麻布，麻線加上狗毛織成的水沙連達戈紋布頗負盛名。

男子服飾

傳統的男子服飾含皮帽、頭飾、胸掛、無袖上衣、腰裙、長套褲、皮革鞋等，與鄒族有點類似。

女子服飾

女子服飾則含頭巾、有袖上衣、胸衣、腰裙、腰帶、膝褲及花草編成的頭冠。服裝顏色以深、淺褐色、藍色、灰色與黑色為多，圖案表現上多為幾何形花紋。

賽夏族

概述

賽夏族的信仰與祭典中，以祖靈與矮靈信仰為主，祖靈信仰與族人關係親近，影響生活運勢、吉凶禍福，也是保護族人的主要力量；矮靈信仰則與古老的歷史傳說有關，以巴斯達隘矮靈祭典最具知名度。

服飾材料來源

賽夏族服飾傳統以麻為主要材料，經過多道程序成為麻布後，再縫接為服裝。

傳統服飾顏色搭配

  1. 傳統服裝形式有無袖長衣、無袖短衣、腰裙、遮陰布、披風、胸兜等，常用白、紅、黑三色，以白色為底、紅色為圖樣、黑色為點綴用色。
  2. 服裝分為日常服裝與儀式慶典服裝，日常服裝多半是素色麻布衣，重要場合時穿戴有幾何花紋圖樣的盛裝，再搭配上各種裝飾品。
  3. 賽夏族的裝飾品有頭飾、頸飾、耳飾、手飾、腿飾等，傳統材料使用貝殼、骨頭、竹子等，現在也加入塑膠、鈕釦、亮片等現代材質飾品。

泰雅族

概述

泰雅族分布在臺灣中北部山區，盛行織布與紋面文化，是臺灣原住民族中分布領域最廣的民族。

服飾材料材料

泰雅族服飾主要材料是麻、毛及棉線，以藍、黃、紅、黑、白色等為主要顏色。

男女服飾

  1. 泰雅族男子服裝有額帶、頭飾、胸兜、無袖長上衣（背心）或有袖長上衣，以及遮陰布與刀帶等。
  2. 女子服裝有額帶、頭飾、長袖對襟短上衣、片裙、護腳布等。

傳統服飾特色

  1. 服裝的圖紋樣式上，衣服正面多使用菱形紋，背面常用複雜花紋，傳說菱形紋是眼睛的象徵，代表祖靈的庇佑。
  2. 泰雅族的傳統服飾-貝珠衣，是結婚時重要的聘禮，婚禮前男生需要送給女生一件到數十件的貝珠衣。貝珠衣為無袖長上衣的形式，白色貝珠橫綴或縱綴在背心上；貝珠衣主要作為寶物珍藏，在重要場合才由頭目、氏族長老、勇士穿著出席。
